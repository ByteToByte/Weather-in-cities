# Weather-in-cities

## Описание
Легкое Погодное Приложение на React.js с использованием Bootstrap 4
Узнай погоду в вашем городе

## Start
Для начала произведем установку пакетов:
```bash
npm init
npm install
```

Далее запустим сервера 
```bash 
cd weather -> npm start 
```

## Используемые технологии 

  - __Html__
 - __Css__
- __React__ 
 - __Bootstrap__
